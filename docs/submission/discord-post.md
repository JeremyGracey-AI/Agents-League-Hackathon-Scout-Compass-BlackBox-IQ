**Compass-BlackBox IQ** — the governance layer for an autonomous agent's *memory & competence* 🧭⬛

Microsoft's stack governs what an agent is *allowed* to do — identity, permissions, actions, grounding. **We govern the layer no one else does: what it knows, did, and learned.**

An agent's entire memory lives as plain Markdown in a git repo the human owns. Every action is an immutable, server-cited decision record — the git log **is** the audit trail. An audit mines those records and drafts new skills from the notes the agent overlooked; a human approves them. **Agents propose, humans promote.** `git revert` rolls back a skill — but the blackbox is append-only: behavior is revertible, history is not.

It grounds across all three Microsoft IQ surfaces — **F.A.M.: Facts (Foundry IQ) · Activity (Work IQ) · Meaning (Fabric IQ)** — read-only, source-tagged, never merged into governed memory.
